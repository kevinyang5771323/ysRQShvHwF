# 微博风格论坛项目简介 java529

## 项目概述

本项目是一款基于微博风格的论坛系统，整合了最新技术栈，提供用户注册、登录、发帖、回帖以及用户管理等丰富功能。

## 技术栈

- 后端：Java Spring Boot
- 前端模板引擎：Thymeleaf
- 数据库：MySQL

## 功能模块

- 用户注册与登录
- 帖子发布与回帖
- 用户管理
  - 用户信息编辑
  - 用户行为管理

## 系统角色

- 普通用户：注册、登录、发帖、回帖
- 管理员：用户管理、内容管理

## 运行环境

- Java 8 或更高版本
- MySQL 5.7 或更高版本

## 部署步骤

1. 准备 MySQL 数据库环境
2. 导入项目提供的数据库文件
3. 部署后端应用（Java Spring Boot）
4. 部署前端文件（Thymeleaf 模板）
